Quick Setup

Connecting Home Assistant.

If you already have Home Assistant and want to connect data, the supported workflow is simple.

Setup Steps
  1. Enter your email through the verification flow, and request a setup token.
  2. Read the gaiangrid_telemetry README and install the integration in Home Assistant.
  3. In the Home Assistant integration, enter your issued enrollment token and select the supported electricity sensors you want to share.
  4. Start publishing telemetry so your site can contribute to the live, aggregated public view.
Home Assistant Inputs

You only enter the enrollment token, your supported sensors, and optionally the fallback intervals.

Why Sharing Helps

Shared telemetry creates useful sensor points for voltage, frequency, and power along low-voltage networks. That improves visibility, supports local coordination, and builds the live public aggregated view without exposing household-level telemetry.

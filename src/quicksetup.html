Quick setup

Connecting Home Assistant.

If you have home assistant all ready and want to setup, the supported workflow is simple:

  1. Enter your email through the email verification flow here, and request an enrollment token.
  2. Read the `gaiangrid_telemetry` README and install the integration in Home Assistant. Open the GitHub repository.
  3. In the Home Assistant integration, enter the issued enrollment token and select the supported electricity sensors you want to share. All data you share is optional.

Home Assistant inputs

You only enter the enrollment token, your supported sensors, and optionally the fallback intervals.

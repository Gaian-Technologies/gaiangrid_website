Quick setup

Enroll whenever, connect Home Assistant when ready.

You do not need to install the integration or even have Home Assistant before enrolling. The supported workflow is simple: enroll whenever, then setup the Gaian Grid Home Assistant integration when ready.

  1. Enter your email through the email verification flow at /enroll.
  2. If you have Home Assistant and are ready to setup, read the `gaiangrid_telemetry` README and install the integration in Home Assistant. Open the GitHub repository.
  3. In Home Assistant, enter the issued enrollment token and select the supported electricity sensors you want to share.

Home Assistant inputs

You only enter the enrollment token, your supported sensors, and optionally the fallback intervals.

Quick setup

Install first, enroll second.

The supported workflow is simple. Read the `ha_telemetry` README, install the integration in Home Assistant, then return here to request your enrollment token.

  1. Read the `ha_telemetry` README and install the integration. Open the GitHub repository.
  2. Return to the enrollment portal and complete the email verification flow at /enroll.
  3. In Home Assistant, use the same site URL you enrolled through and the issued enrollment token.

Home Assistant inputs

The setup flow only needs the site URL and the enrollment token.
